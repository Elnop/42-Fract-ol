# Fract'ol

First graphic project on 42 cursus

## Download

```shell
git clone --recurse-submodules https://github.com/Elnop/42-Fract-ol.git

make fractol
```

## Start
```shell
./fractol ( julia [C_R, C_I] ) | mandelbrot | burning_ship
```
*C_R: constant real part  
C_I: constant partie imaginary*

## Controls

- **Scroll:** + / - Zoom  
- **Arrows:** move image  
- **Z:** - Iterations  
- **X:** + Iterations
